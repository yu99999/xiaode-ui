# 安装

该项目已发布到 [npm](https://www.npmjs.com/package/xiaode-ui)

```
npm install xiaode-ui --save
```



# 使用

引入样式

```
import "xiaode-ui/dist/index.css"
```

引入组件

```
import { Button } from "xiaode-ui"
```
